# User Flow

# BayWel User Flow

# Primary Experience

## Step 1 — Physical Interaction

User selects a physical mindfulness card.

The card itself provides standalone value.

No technology required.

---

## Step 2 — Optional Digital Continuation

Card includes subtle QR prompt.

Example:
"Continue this reflection"

User scans QR code.

---

## Step 3 — Immediate Reflection Context

Mobile web page opens instantly.

No:
- app install
- signup wall
- onboarding sequence

The page immediately displays:
- card prompt
- optional reflection guidance
- reflection input choices

---

# Reflection Options

Users may choose:
- silent reflection
- text reflection
- guided AI reflection
- voice reflection (future)

---

# Guided AI Reflection

AI should:
- ask short questions
- encourage awareness
- mirror patterns gently

AI should NOT:
- overwhelm users
- dominate interaction
- become verbose

---

# Anonymous Usage

Users can:
- complete reflections anonymously
- leave without creating accounts
- experience value before signup

This phase establishes trust.

---

# Opt-In Memory Prompt

After repeated engagement:
Display:

"Would you like BayWel to remember your reflections over time?"

Value proposition:
- emotional continuity
- pattern awareness
- growth tracking

NOT:
- data collection
- marketing

---

# Account Creation

Low-friction auth only:
- Apple
- Google
- magic link

No passwords.

---

# Consent Flow

Users choose:
- save reflections
- allow anonymized product insights
- enable personalization

Consent must be:
- explicit
- granular
- reversible

---

# Reflection Memory

Once enabled:
BayWel can:
- remember themes
- identify recurring patterns
- personalize reflection guidance
- provide longitudinal summaries

---

# Longitudinal Experience

Examples:
- weekly reflection summaries
- recurring emotional themes
- suggested next card categories
- growth observations

The tone should remain:
- calm
- reflective
- supportive

---

# Creator Flow

Creators may:
- pull cards on video
- scan QR live
- share reflection experiences
- demonstrate AI-guided journeys

This creates:
- authentic content

- organic product demonstration
